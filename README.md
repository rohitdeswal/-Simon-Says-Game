# 🎮 Simon Says Game

A simple memory game built using HTML, CSS, and JavaScript.

---

## 📌 Description

This game tests your memory skills.  
The system generates a sequence of colors, and you have to repeat the same sequence correctly.  
Each level adds a new color to the sequence.

---

## 🕹️ How to Play

- Press any key to start the game  
- Watch the color that flashes  
- Click the same color  
- Repeat the full sequence  
- Each level adds a new color  
- Wrong click = Game Over  

---

## 🚀 Features

- Random color sequence  
- Increasing difficulty  
- Score display  
- Simple UI  
- Restart option  

---

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript  

---

## 📂 Files

- index.html  
- style.css  
- app.js  

---

## ▶️ How to Run

1. Download or clone the project  
2. Open `index.html` in your browser  
3. Press any key to start  

---

## 💡 Note

Game logic is handled using JavaScript functions like sequence generation, input checking, and reset.
