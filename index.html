Simon Says Game

Press any key to start the game

Start: Press any key to begin.
Watch: The game flashes a random color (e.g., Red).
Repeat: Click the same color that just flashed.
Memorize: The game adds a new color to the end (e.g., Red → Blue).
Match: Click the entire sequence in the exact order.
Level Up: Every time you succeed, the sequence gets one step longer.
Game Over: If you click the wrong color, the game ends and resets
